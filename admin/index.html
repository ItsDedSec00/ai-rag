Verwendung

So bindest du RAG-Chat in andere Anwendungen ein.

Python (openai SDK)
from openai import OpenAI

client = OpenAI(
    api_key="rck-...",
    base_url="http://<server>:8080/v1"
)

resp = client.chat.completions.create(
    model="llama3.2:1b",
    messages=[{"role": "user", "content": "Was ist RAG?"}]
)
print(resp.choices[0].message.content)
curl
curl http://<server>:8080/v1/chat/completions \
  -H "Authorization: Bearer rck-..." \
  -H "Content-Type: application/json" \
  -d '{"model":"llama3.2:1b","messages":[{"role":"user","content":"Hallo"}]}'
